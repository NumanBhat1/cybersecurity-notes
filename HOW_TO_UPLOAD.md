# How to upload this to GitHub

Same process as your other repos.

1. https://github.com/new -> name `cybersecurity-notes` -> Public ->
   don't add README/license -> Create.
2. Click **"uploading an existing file"**.
3. Open this `cybersecurity-notes` folder, select everything **inside** it
   (the `networking`, `linux`, `windows`, `siem`, `mitre` folders + README.md, LICENSE),
   and drag it in. (Skip this HOW_TO_UPLOAD.md file.)
4. Commit.

## After uploading

- Add topics (gear next to About): `cybersecurity`, `notes`, `soc`, `study`, `security-plus`.
- Pin it (you can pin up to 6 repos).

## Keep adding to it

This is meant to grow as you study. Good next notes to add:
- Active Directory basics
- Common attacks (phishing, malware types, social engineering)
- Cryptography basics (hashing vs encryption)
- Incident response steps
Just create a new `.md` in the right folder and add a row to the README table.
