# Security+ Acronyms

The exam is heavy on acronyms. This is the high-frequency set worth knowing cold.

## Core security / CIA

| Acronym | Meaning |
|---------|---------|
| CIA | Confidentiality, Integrity, Availability |
| AAA | Authentication, Authorization, Accounting |
| IAM | Identity and Access Management |
| MFA | Multi-Factor Authentication |
| SSO | Single Sign-On |
| PII | Personally Identifiable Information |
| PHI | Protected Health Information |
| RBAC | Role-Based Access Control |
| ABAC | Attribute-Based Access Control |
| DAC | Discretionary Access Control |
| MAC | Mandatory Access Control |
| PoLP | Principle of Least Privilege |

## Crypto

| Acronym | Meaning |
|---------|---------|
| AES | Advanced Encryption Standard (symmetric) |
| DES / 3DES | Data Encryption Standard (legacy) |
| RSA | Rivest-Shamir-Adleman (asymmetric) |
| ECC | Elliptic Curve Cryptography |
| PKI | Public Key Infrastructure |
| CA | Certificate Authority |
| CSR | Certificate Signing Request |
| CRL | Certificate Revocation List |
| OCSP | Online Certificate Status Protocol |
| TLS / SSL | Transport Layer Security / Secure Sockets Layer |
| HMAC | Hash-based Message Authentication Code |
| SHA | Secure Hash Algorithm |
| MD5 | Message Digest 5 (weak hash) |
| PFS | Perfect Forward Secrecy |

|---------|---------|
| DoS / DDoS | Denial of Service / Distributed DoS |
| MITM | Man-in-the-Middle (on-path attack) |
| XSS | Cross-Site Scripting |
| CSRF | Cross-Site Request Forgery |
| SQLi | SQL Injection |
| RCE | Remote Code Execution |
| APT | Advanced Persistent Threat |
| IoC | Indicator of Compromise |

## Operations / governance

| Acronym | Meaning |
|---------|---------|
| SIEM | Security Information and Event Management |
| SOAR | Security Orchestration, Automation, and Response |
| SOC | Security Operations Center |
| DLP | Data Loss Prevention |
| EDR / XDR | Endpoint / Extended Detection and Response |
| GRC | Governance, Risk, and Compliance |
| BIA | Business Impact Analysis |
| BCP / DRP | Business Continuity / Disaster Recovery Plan |
| RTO | Recovery Time Objective |
| RPO | Recovery Point Objective |
| MTTR | Mean Time To Repair |
| MTBF | Mean Time Between Failures |
| SLA | Service Level Agreement |
| SLE / ALE / ARO | Single Loss Expectancy / Annual Loss Expectancy / Annual Rate of Occurrence |

## Risk formulas to remember

- **SLE = Asset Value x Exposure Factor**
- **ALE = SLE x ARO**

(These show up in quantitative risk questions.)
